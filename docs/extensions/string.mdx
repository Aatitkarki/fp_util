# String Extensions

### Usage
Add the following import to your Dart file:
```dart
import 'package:fp_util/fp_util.dart';
```

### isNullOrEmpty
```dart title="check if a string is null or empty"
final String? str = null;
print(str.isNullOrEmpty); // true
```

### isNotNullNotEmpty
```dart title="check if a string is not null or not empty"
final String? str = 'hello';
print(str.isNotNullNotEmpty); // true
```

### isEmail
```dart title="check if a string is a valid email"
final String email = 'test@gmail.com';
print(email.isEmail); // true
```

### getOrDefault
```dart title="get a default value if the string is null or empty"
final String? str = null;
print(str.getOrDefault('default')); // default
```

### getOrNull
```dart title="get a null if the string is null or empty"
final String? str = null;
print(str.getOrNull()); // null
```

### fileNameFromUri
```dart title="get the file name from a uri"
final String uri = 'https://example.com/file.txt';
print(uri.fileNameFromUri); // file.txt
```

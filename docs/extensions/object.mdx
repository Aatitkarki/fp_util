# Object Extensions

### Usage
Add the following import to your Dart file:
```dart
import 'package:fp_util/fp_util.dart';
```

### let
```dart title="let"
void main() {
  final result = 5.let((it) => it * 2);
  print(result); // 10
}
```

### also
```dart title="also"
void main() {
  final result = 5.also((it) => print(it+5));
  // 10
  print(result); // 5
}
```

### safeCast
```dart title="safeCast"
void main() {
  final result = 5.safeCast<String>();
  print(result); // null
}
```

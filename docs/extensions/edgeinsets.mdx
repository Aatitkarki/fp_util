# EdgeInsets Extensions
Extensions for EdgeInsets

## Usage
Add the following import to your Dart file:
```dart
import 'package:fp_util/fp_util.dart';
```

### addTop
```dart title="add value in top of EdgeInsets"
EdgeInsets edgeInsets = EdgeInsets.all(10);
EdgeInsets newEdgeInsets = edgeInsets.addTop(10);
```

### addRight
```dart title="add value in right of EdgeInsets"
EdgeInsets edgeInsets = EdgeInsets.all(10);
EdgeInsets newEdgeInsets = edgeInsets.addRight(10);
```

### addBottom
```dart title="add value in bottom of EdgeInsets"
EdgeInsets edgeInsets = EdgeInsets.all(10);
EdgeInsets newEdgeInsets = edgeInsets.addBottom(10);
```

### addLeft
```dart title="add value in left of EdgeInsets"
EdgeInsets edgeInsets = EdgeInsets.all(10);
EdgeInsets newEdgeInsets = edgeInsets.addLeft(10);
```

### addX
```dart title="add value in left and right of EdgeInsets"
EdgeInsets edgeInsets = EdgeInsets.all(10);
EdgeInsets newEdgeInsets = edgeInsets.addX(10);
```

### addY
```dart title="add value in top and bottom of EdgeInsets"
EdgeInsets edgeInsets = EdgeInsets.all(10);
EdgeInsets newEdgeInsets = edgeInsets.addY(10);
```

### removeTop

### Usage
Add the following import to your Dart file:
```dart
import 'package:fp_util/fp_util.dart';
```


### Theme
```dart title="Theme"
final theme = context.theme;
```

### Color Scheme
Extensions for color scheme

```dart title="Color Scheme"
final colorScheme = context.colorScheme;
```

```dart title="Primary Colors"
final primaryColor = context.primaryColor;
final onPrimaryColor = context.onPrimaryColor;
final primaryContainerColor = context.primaryContainerColor;
final onPrimaryContainerColor = context.onPrimaryContainerColor;
```

```dart title="Secondary Colors"

final onSecondaryColor = context.onSecondaryColor;
final secondaryContainerColor = context.secondaryContainerColor;
final onSecondaryContainerColor = context.onSecondaryContainerColor;
```

```dart title="Tertiary Colors"

final onTertiaryColor = context.onTertiaryColor;
final tertiaryContainerColor = context.tertiaryContainerColor;
final onTertiaryContainerColor = context.onTertiaryContainerColor;
```

```dart title="Surface Colors"
final surfaceColor = context.surfaceColor;
```

### Size
Extensions for size


### Focus
Extensions for focus

```dart title="Focus"
// Remove focus
context.removeFocus();
// Check if the context has focus
final hasFocus = context.hasFocus;
```

### Responsive Value
Extensions for responsive value

```dart title="Responsive Value"
final responsiveValue = context.responsiveValue<int>(
  20,
  tablet: 50,
  desktop: 25,
);
```

### Themed Value
Extensions for themed value

```dart title="Themed Value"

  20,

);
```

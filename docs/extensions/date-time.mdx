# DateTime Extensions
Extensions for date time

## Usage
To use this package, just import it into your file and use the extensions.
```dart
import 'package:fp_util/fp_util.dart';
```

### ymd
```dart title="Format date to yyyy-MM-dd"
final formattedDate = DateTime.now().ymd;
```

### jms
```dart title="Format date to HH:mm:ss"
final formattedTime = DateTime.now().jms;
```

### yyMMdd
```dart title="Format date to yyMMdd"
final formattedDate = DateTime.now().yyMMdd();
```

### yMEd
```dart title="Format date to EEEE, M/d/y"
final formattedDate = DateTime.now().yMEd();
```

### jm
```dart title="Format date to h:mm"
final formattedTime = DateTime.now().jm();
```

### format
```dart title="Format date to custom format"
final formattedDate = DateTime.now().format('yyyy-MM-dd');
```

### addDays
```dart title="Add days to date"
final newDate = DateTime.now().addDays(5);
```

### addHours
```dart title="Add hours to date"
final newDate = DateTime.now().addHours(5);
```

### addMinutes
```dart title="Add minutes to date"
final newDate = DateTime.now().addMinutes(5);
```

### addSeconds
```dart title="Add seconds to date"
final newDate = DateTime.now().addSeconds(5);
```

### subtractDays
```dart title="Subtract days from date"
final newDate = DateTime.now().subtractDays(5);
```

### subtractHours
```dart title="Subtract hours from date"
final newDate = DateTime.now().subtractHours(5);
```

### subtractMinutes
```dart title="Subtract minutes from date"
final newDate = DateTime.now().subtractMinutes(5);
```

### subtractSeconds
```dart title="Subtract seconds from date"
final newDate = DateTime.now().subtractSeconds(5);
```

### formatParse
```dart title="Parse date from custom format"
final date = DateTimeUtils.formatParse('2024-01-01','yyyy-MM-dd');
```

### timestamp
```dart title="Get timestamp from date"
final timestamp = DateTime.now().timestamp;
```

### timeAgo
```dart title="Get time ago from date"
final timeAgo = DateTime.now().subtractDays(5).timeAgo();
```

### fiscalYear
```dart title="Get fiscal year from date"
final fiscalYear = DateTime.now().fiscalYear(4 //start month for fiscal year);
```

### currentFiscalYear
```dart title="Get current fiscal year"
final currentFiscalYear = DateTimeUtils.currentFiscalYear(4 //start month for fiscal year);
```

### isToday
```dart title="Check if date is today"
final isToday = DateTimeUtils.isToday(DateTime(2024, 1, 1));
```

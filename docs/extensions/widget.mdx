# Widget Extensions

### Usage
Add the following import to your Dart file:
```dart
import 'package:fp_util/fp_util.dart';
```


### scrollable
```dart title="Wraps a widget with a SingleChildScrollView"
  Column(
    children: [
      Text('Scrollable 1'),
      Text('Scrollable 2'),
    ],
  ).scrollable(),
```


### clickable
```dart title="Wraps a widget with a GestureDetector"
  Text('Clickable').clickable(() {
    print('clicked');
  }),
```

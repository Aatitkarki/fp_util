# File Extensions
Extensions for file

### Usage
Add the following import to your Dart file:
```dart
import 'package:fp_util/fp_util.dart';
```

### fileName
```dart title="get file name"
final file = File('path/to/file.txt');
final name = file.fileName;
print(name); // file.txt
```

### extension
```dart title="get file extension"
final file = File('path/to/file.txt');
final ext = file.extension;
print(ext); // txt
```

### mimeType
```dart title="get file mime type"
final file = File('path/to/file.txt');
final mime = file.mimeType;
print(mime); // text/plain
```

### isImage

# Constants

### Usage
Add the following import to your Dart file:
```dart
import 'package:fp_util/fp_util.dart';
```

### Boolean
```dart title="boolean constants"
const bool kTrue = BooleanConst.TRUE;
const bool kFalse = BooleanConst.FALSE;
```

```dart title="Spacing Example"
  /// vertical space
  Column(
    children:[
      Text('Hello',),
      Sizes.gap4,
      Text('World',),
    ]
  );

  /// horizontal space
  Row(
    children:[
      Text('Hello',),
      Sizes.p4,
      Text('World',),
    ]
  );
```

```dart title="Sliver Spacing Example"

    slivers: [
```

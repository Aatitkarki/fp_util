### Conditional Widget
Widget used to display a child based on a condition
```dart
import 'package:fp_util/fp_util.dart';
ConditionalWidget(
  condition: true,
  trueChild: Text('This is displayed'),
  falseChild: Text('This is not displayed'),
)
```

### HideKeyboard
Widget used to hide the keyboard when tapped on screen
```dart
import 'package:fp_util/fp_util.dart';
HideKeyboard(
  child: TextField(),
)
```

### PlatformWidget
Widget used to display different widgets based on the platform
```dart
import 'package:fp_util/fp_util.dart';
class MyWidget extends PlatformWidget<Text,Text> {
  @override
  Text buildMaterialWidget(BuildContext context) {
    return Text('Android');
  }
  @override
  Text buildCupertinoWidget(BuildContext context) {
    return Text('iOS');
  }
}
```

### ImagePickerButton


### ScaffoldWrapper
Widget used to wrap a scaffold with a hide keyboard and safe area
```dart title="ScaffoldWrapper"
ScaffoldWrapper(
  appBar: AppBar(
    title: Text('Title'),
  ),
  body: Center(
      child: Text('Body'),
    ),
)
```

# Input Formatters

### Usage
Add the following import to your Dart file:
```dart
import 'package:fp_util/fp_util.dart';
```

### UpperCaseTextFormatter
```dart title="formats text to upper case"
TextField(
  inputFormatters: [
    UpperCaseTextFormatter(),
  ],
)
```

### LowerCaseTextFormatter
```dart title="formats text to lower case"
TextField(
  inputFormatters: [
    LowerCaseTextFormatter(),
  ],
)
```

### SentenceCaseTextFormatter
```dart title="formats text to sentence case"
TextField(
  inputFormatters: [
    SentenceCaseTextFormatter(),
  ],
)
```

### TitleCaseTextFormatter
```dart title="formats text to title case"
TextField(
  inputFormatters: [
    TitleCaseTextFormatter(),
  ],
)
```

### DecimalTextFormatter
```dart title="formats decimal with 2 decimal places"
TextField(
  inputFormatters: [
    DecimalTextFormatter(2),
  ],
)
```

### EmojiTextFormatter
```dart title="formats text to remove emojis"
TextField(
  inputFormatters: [
    EmojiTextFormatter.deny(),
  ],
)
```

```dart title="formats text to allow emojis"
TextField(
  inputFormatters: [
    EmojiTextFormatter.allow(),
  ],
)
```


## Masked Input Formatters

### custom
```dart title="formats text to custom mask"
TextField(
  inputFormatters: [
    MaskedTextFormatter.custom(
      mask: '0000-0000-0000-0000',
      filter: {"0": RegExp(r'[0-9]')},
    ),
  ],
)
```

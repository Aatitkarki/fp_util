percent(int percent) → Color
Returns the Color with the opacity set to percent. The percent value must be between 0 and 100.

Static Methods

fromHex(String hex, [int opacity = 100]) → Color
Returns the Color from a hex string. The hex value can be 6 or 7 characters long. If the hex value is 7 characters long, the first character is assumed to be the opacity. The opacity value must be between 0 and 100. If the opacity value is not provided, it defaults to 100.
